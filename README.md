## Projects

### [Credit Card Fraud Detection](./Credit-Card-Fraud-Detection) — End-to-End ML
Fraud detection on a highly imbalanced dataset (0.58% fraud). Tuned **XGBoost** lifts PR-AUC from 0.18 → **0.88** and catches **90% of fraud** while flagging only **0.35%** of legitimate transactions. Covers EDA, leak-safe feature engineering, model comparison, hyperparameter tuning, **SHAP** explainability, and a business-impact report.

**Stack:** Python · XGBoost · SHAP · scikit-learn · imbalanced-learn
